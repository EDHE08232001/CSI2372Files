How to run ?

> Compile all the cpp in the project with ' clang++ *.cpp -std=c++11 -o project '
> Execute the generate ' .exe ' file 

> The .txt files Saved-Deck.txt, Saved-DiscardPile.txt, Saved-P1.txt, Saved-P2.txt, Saved-TradeArea.txt contain
the information about the saved game. 
> The initial submitted .txt files contain a saved game that can be reloaded at the beginning. 
> These files can be overriden when saving a new game.
